# 可以直接import的laydate

laydate是一个非常优秀的日期选择组件，这里只是稍加修改，使其可以支持es6 import方式引入

-   使用
    ```javascript
    import laydate from "laydae-es6"
    laydate.render({
        elem:"#datePicker"
    })
    ```
